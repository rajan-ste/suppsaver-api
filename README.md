# Suppsaver API

## Setup
#### Clone the repo
```
git clone https://github.com/rajan-ste/suppsaver
```

#### Install dependencies
```
cd api
npm install
```

#### Environmental Variables
```
PORT=
JWT_KEY=
DB_HOST=
DB_USER= 
DB_PASSWORD= 
DB_NAME= 
API_KEY= 
```

#### Run
Development:
```
npm run dev
```
Production:
```
npm run start
```
